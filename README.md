# 智能旅游系统简介 python1271

## 项目概述


## 技术栈

- **后端框架：** Django
- **编程语言：** Python 3.7/3.8
- **前端框架：** Vue.js
- **数据库：** MySQL 5.7 或更高版本
- **数据库工具：** Navicat 11（推荐）
- **开发工具：** PyCharm/VS Code

## 功能模块

系统主要包含以下功能模块：

- 旅游景点查询与预订
- 酒店查询与预订
- 旅游路线查询与预订
- 用户账户管理
- 订单管理

## 系统角色

- **普通用户：** 可进行景点、酒店、路线查询与预订
- **管理员：** 负责管理用户、订单以及系统维护

## 运行环境

- Python 3.7 或 3.8
- Django 框架
- Vue.js 前端框架
- MySQL 5.7 或更高版本
- Navicat 11（可选，用于数据库管理）

## 部署步骤

1. 环境准备：确保Python、MySQL等依赖项已正确安装
2. 克隆或下载项目代码
3. 安装项目依赖：通过pip安装requirements.txt文件中的依赖包
4. 数据库配置：使用Navicat或命令行创建数据库并导入初始数据
5. 运行项目：根据开发文档启动后端和前端服务

## 目录结构

```
project/
├── backend/              # 后端代码目录
├── frontend/             # 前端代码目录
└── scripts/              # 数据库初始化脚本等
```

## 核心亮点

- **模块化开发：** 方便后期的功能扩展与维护
- **高效的后端处理：** 利用Django框架优势，快速处理请求，提高响应速度
- **数据库兼容性：** 支持多种版本的MySQL，灵活应对不同环境需求
